# spotify-clone
📌 Project Description

This is a Spotify-inspired web music player built using HTML, CSS, and JavaScript.
The project replicates the core functionality of a music streaming platform, including song selection, play/pause controls, progress tracking, and next/previous navigation.

The goal of this project was to strengthen my understanding of DOM manipulation, event handling, and audio control in JavaScript.

🚀 Features

🎧 Play and Pause songs

⏭ Next and Previous controls

📀 Dynamic song list rendering

🎵 Real-time progress bar update

⏩ Seek control (jump to specific part of song)

🎬 Playing animation (GIF effect)

📱 Basic responsive design

🛠 Technologies Used

HTML5 – Structure

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Functionality & Audio control

Font Awesome – Icons
